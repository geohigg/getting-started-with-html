HTML Basics:

HTML, Hyper Text Markup Language is comprised of:
  • Text content - ie what you see.

  • Markup - the construction blocks of html that define what the text looks like.

  • References to other documents (Links to video, and images).

  • Links to other pages.
Overview of HTML Markup:

HTML is comprised of tags which defines the behavior and appearance of a webpage. Tags are structured as follows:
<name> content </name>.

  • <name> - is called the opening tag.
  • contents - is called the tag contents.
  • </name> - is called the closing tag.

    • The set of tags is also referred too as an Element. It is important to remember that the behavior of a tag will
    remain in effect until it is closed </>!

    Example elements found in this exercise
    • <b>Will make text bold </b>

    • <em>Will make text italic </em>

    • <a> </a> is an anchor tag. An example of this tag in use is in the creation of a link to another webpage.
      That tag is constructed with the following syntax: <a href = "url"> URL Plain text </a>. In use it would look
      something like this: G+ Community Homepage!

    • <img src = "URL" alt = "alternate text"> Will embed an image here.
      • Note: A closing tag is not needed for this element. It is whats referred to as a Void Tag.
    Notes on Whitespace

    There are two types of whitespace
    1. Inline
    2. Block
    Inline Whitespace: Ends a line by wrapping text to a new line.

    Examples of inline elements are:
    • a
    • span
    • br
    • img
    • strong

    Block Whitespace: Adds an invisible box around the text encapsulated by the tag

    Examples of inline elements are:
    • p
    • div
    • form


    The Structure of an HTML Document:

    The bare bones markup of an html document looks like this...

    <!DOCTYPE HTML>
    <html>
          <head>
                <title> TITLE </title>
          </head>
          <body>
                <content> the content (text, images, links, etc) is contained in the HTML body </content>
          </body>
    </html>
